Projeto feito para exercitar minhas capacitações em HTML, CSS e JAVASCRIPT.
slogan: O mais puro sabor tradicional brasileiro.

O projeto é composto por Home, Contato, Produtos, Serviços.

Home: Sobre a empresa, objetivo, vídeo de demostração, analise de clientes, rodapé

Contato: informações de contato, e um form para enviar email, além das redes sociais.

Produtos: lista de produtos da Marca CafeStore Brasil, café expresso, forte, preto, cappucino...

Serviços: Contato empresarial para fornecimento serviços BtoB.
